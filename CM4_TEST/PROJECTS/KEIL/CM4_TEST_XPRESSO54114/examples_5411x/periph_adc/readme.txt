LPC5411x Simple ADC example
===========================

Example description
-------------------
The ADC example uses the internal temperature sensor as an input to channel 0 and converts the analog voltage to a digital value. The conversion is done at 3Hz and the output value is display on the UART debug port.

Special connection requirements
-------------------------------
There are no special connection requirements for this example.
