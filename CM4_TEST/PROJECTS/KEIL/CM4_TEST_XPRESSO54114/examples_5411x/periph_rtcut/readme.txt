LPC5411x Real-Time clock example with Universal Time conversion
===============================================================

Example description
-------------------
This example shows how to setup and use the RTC with the Universal Time
driver to generate readable time structures. The alarm will trigger
every 5 seconds waking up the device and display the current time.

Special connection requirements
-------------------------------
There are no special connection requirements for this example.

Build procedures:
-----------------
Visit the at 'LPCOpen quickstart guides' [http://www.lpcware.com/content/project/lpcopen-platform-nxp-lpc-microcontrollers/lpcopen-v200-quickstart-guides]
