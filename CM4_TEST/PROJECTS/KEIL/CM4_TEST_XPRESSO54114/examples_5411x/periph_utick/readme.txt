LPC5411x UTick example
======================

Example description
-------------------
This example shows how to setup and use the UTick to generate periodic
interrupts.

This example configures UTick to generate interrupts every second.
The board LED 0 is toggled each time the interrupt occurs.

Special connection requirements
-------------------------------
There are no special connection requirements for this example.
