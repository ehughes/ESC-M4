DESCRIPTION
===========

Flashloader for the LPC54xxx

CONFIGURATION
=============

The project contains configuration:

- FlashNXPLPC54xxxM4F_RAM64K
Builds a flashloader for the LPC54xxx Cortex M4F core
Output file: FlashNXPLPC54xxxM4F_RAM64K.out
Macro file: FlashNXPLPC54xxxM4F.mac

- FlashNXPLPC54xxM0PL_RAM64K
Builds a flashloader for the LPC54xxx Cortex M4F core
Output file: FlashNXPLPC54xxxM0PL_RAM64K.out
Macro file: FlashNXPLPC54xxxM0PL.mac

USAGE
=====
The flash loader assumes that the application code is linked on address 0x00000000.
