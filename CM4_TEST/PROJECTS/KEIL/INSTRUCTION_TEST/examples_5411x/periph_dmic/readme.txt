LPC5411x DMIC example
=====================

Example description
-------------------
The DMIC example shows how to use the digital microphone interface to capture audio from pdm mic.
Captured audio is also transmitted via uart.
Hardware voice activity detector function is demonstrated by the red led on the board.

Special connection requirements
-------------------------------
A sensor shield board is required with the board to use this example.
